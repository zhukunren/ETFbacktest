# 前端优化文档

## 优化目标
根据设计原型图 `frontend-prototype.png` 对前端界面进行优化，使其更加简洁、现代、符合设计规范。

## 主要优化内容

### 1. 整体布局优化
- **左侧配置面板宽度调整**：从 454px 缩小到 400px，使布局更紧凑
- **间距优化**：工作区间距从 9px 增加到 16px，视觉更舒适
- **卡片圆角**：统一使用 12px 圆角，更加现代
- **阴影优化**：使用更轻微的阴影 `0 1px 3px rgba(0, 0, 0, 0.05)`

### 2. 顶部导航栏优化
- **高度调整**：从 68px 减小到 64px
- **标题字体**：从 24px/800 调整为 20px/700，更符合现代设计
- **Logo图标**：从圆形改为圆角矩形（8px），尺寸从 38px 调整为 36px
- **统计卡片**：
  - 背景色更浅 (`#F9FAFB`)
  - 边框颜色 `#E5E7EB`
  - 圆角从 8px 改为 6px
  - 高度从 44px 减小到 38px
  - 字体大小优化：标签 13px，数值 14px
- **按钮高度**：从 44px 调整为 38px

### 3. 配置面板优化
- **区块样式**：去除独立的边框和阴影，改用底部分割线
- **区块间距**：从 8px 改为 0，使用底部边框分隔
- **内边距**：统一为 20px 18px
- **标题字体**：从 18px/800 调整为 16px/700

### 4. 结果面板优化
- **标签栏样式**：
  - 默认标签颜色 `#6B7280`，激活色 `#2563EB`
  - 激活指示器：高度从 3px 减小到 2px，圆角从 999px 改为 2px
  - 添加 hover 效果
- **指标卡片**：
  - 布局改为 3 列（原来是 6 列）
  - 圆角从 8px 增加到 12px
  - 最小高度从 150px 减小到 130px
  - 内边距增加到 20px
  - 使用 flex 布局替代 grid，更好控制间距
  - 标题颜色更浅 `#6B7280`，字体 14px/600
  - 数值字体从 27px/850 调整为 32px/700
  - 图标尺寸从 44px 减小到 40px，圆角从 999px 改为 10px
- **内容区块**：去除独立边框，使用透明背景
- **间距优化**：面板内边距从 13px 增加到 20px，区块间距从 14px 增加到 20px

### 5. 响应式保持
- 保持原有的响应式断点（1500px, 980px, 640px）
- 在小屏幕上指标卡片自动调整为 2 列或 1 列

## 设计原则

1. **简洁优先**：去除不必要的边框和阴影
2. **间距统一**：使用一致的间距系统（6px, 8px, 12px, 16px, 20px）
3. **字重适中**：避免过重的字体（700 替代 800/850）
4. **色彩克制**：使用更浅的灰色系，减少视觉噪音
5. **圆角现代**：统一圆角风格，卡片 12px，小组件 6-10px

## 技术细节

### 关键文件修改
1. `frontend/src/App.vue` - 主布局和顶部导航栏
2. `frontend/src/components/BacktestConfigPanel.vue` - 左侧配置面板
3. `frontend/src/components/BacktestResults.vue` - 右侧结果展示面板

### CSS 变量使用
- `--surface`: 背景白色
- `--border`: 边框颜色 #E5E7EB
- `--text`: 主文本色
- `--muted`: 次要文本色
- `--accent`: 强调色 #2563EB
